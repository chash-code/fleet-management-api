# Fleet Management API

Backend API for fleet management system built with Node.js, Express, and PostgreSQL (Supabase).

## Features
- User management (customer, owner, driver roles)
- Vehicle management with driver assignment
- Trip booking system with cost calculation
- Analytics endpoint
- Rate limiting on vehicle creation
- Request logging

## Tech Stack
- Node.js
- Express.js
- PostgreSQL (Supabase)
- Environment variables (.env)
